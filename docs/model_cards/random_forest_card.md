# Model Card: Random Forest Classifier

## Model Details

- **Architecture**: Scikit-learn `RandomForestClassifier`
- **Hyperparameters**: `n_estimators=200`, `max_depth=None`, `min_samples_leaf=2`, `class_weight="balanced"`, `random_state=42`
- **Training script**: `scripts/bootstrap_artifacts.py`
- **Artifact**: `models/random_forest.pkl`

## Intended Use

Multi-class network intrusion detection on MinMax-scaled flow features derived from CICIDS-2017. Primary deployment target: SOC analyst dashboard with real-time SHAP explanations.

## Dataset

- **Source**: CICIDS-2017 (synthetic balanced version generated by `scripts/generate_samples.py`)
- **Classes**: 14 (1 BENIGN + 13 attack types)
- **Features**: 78 flow-level statistics
- **Training samples**: ~28,000 (2,000 × 14 classes)
- **Test samples**: ~5,600 (400 × 14 classes)

## Limitations

- Trained on synthetic data — performance on real CICIDS-2017 or live traffic will differ.
- Class distributions are balanced; real traffic is heavily skewed toward BENIGN.
- No temporal modelling — each flow is classified independently.

## Caveats & Recommendations

- Retrain on real CICIDS-2017 raw CSV before production deployment.
- Apply class-prior calibration or threshold tuning for deployment-specific FPR targets.
